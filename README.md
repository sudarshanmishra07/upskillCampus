# Quiz Game

## Project Description

This project is a simple Quiz Game developed using Python as a part of my Industrial Internship. The main purpose of this project is to test users with multiple-choice questions and display their final score after completing the quiz.

The project uses basic Python concepts such as variables, loops, conditional statements, lists, dictionaries and user input. It was developed to improve my programming and problem-solving skills.

## Features

- User enters their name before starting the quiz.
- Displays instructions for the quiz.
- Contains 10 multiple-choice questions.
- Accepts only valid options (A, B, C or D).
- Shows whether the selected answer is correct or wrong.
- Calculates score and percentage.
- Displays grade and remarks.
- Shows pass or fail result.
- Provides an option to play again.

## Technologies Used

- Python 3
- Visual Studio Code

## How to Run

1. Download or clone this repository.
2. Open the project in Visual Studio Code.
3. Run the quiz_game.py file using Python.
4. Answer all the questions.
5. View your final result.
